# chorntile

2D **стратегія-виживалка з автобоєм** у пост-апокаліптичному світі, де аномальні зони розрослися по всьому світу. Тайлова база, центральна **Сільрада**, бензиновий **Генератор**, призначення вцілілих на роботи, герої з перками, хвилі мутантів і вилазки. Референс механіки — *Tiles Survive!* (FunPlus); тон — *Frostpunk*.

> **Статус:** ранній прототип, Phase 1 — Milestone **M0: Фундамент**.

## Стек

- **Движок:** Godot **4.6** (Forward+, Jolt Physics)
- **Мова:** GDScript (зі статичною типізацією)
- **Платформа:** Windows (десктоп)

## Фази розробки

1. **Phase 1 (MVP) — single-player**: одне поселення (село Глуш), ~3 год геймплею, 4 герої, Сільрада Lv2, хвилі мутантів.
2. **Phase 2** — розширення героїв, тактичні експедиції, 2 фракції.
3. **Phase 3** — глобальна карта Пустки, SP 4X з AI-дипломатією.
4. **Phase 4 (опціональна)** — мультиплеєр + альянси + сервер. Рішення ухвалюємо **після** Phase 3.

## Як запустити

1. Встанови [Godot 4.6](https://godotengine.org/download) (standard, не .NET).
2. Клонуй репо:
   ```powershell
   git clone https://github.com/stovburtm-web/chorntile.git
   ```
3. У Godot → *Import* → обери `project.godot`.
4. `F5` — запуск проєкту.
