# `chorntile` — Project Plan

> Дизайн-документ + дорожня карта реалізації. Живий файл — оновлюємо по ходу.
> **Статус:** Phase 1 / Milestone M0.

## 0. TL;DR

**chorntile** — 2D **стратегія-виживалка з автобоєм** у сеттінгу Зони відчуження.
Головний референс механіки: **Tiles Survive!** (FunPlus) — тайлова база, центральна будівля-"серце", призначення вцілілих на роботи, герої з перками, експедиції в дикі землі, пізніше альянси.
Додаткові орієнтири: *State of Survival*, *Whiteout Survival*, *Frostpunk* (для тону), *They Are Billions* (для хвиль оборони).

### Лор у трьох реченнях

1. Після **Розлому** (глобальний катаклізм у недалекому майбутньому — ядерна війна + каскад аномальних Вибухів, див. §5) аномальні зони розрослися по всьому світу. Стара географія зникла; людство живе плямами серед нової **Пустки**.
2. Ти прокидаєшся єдиним вижилим у покинутому **селі Глуш** у глухому лісовому краї — подалі від великих аномалій. У центрі села стоїть занедбана **Сільрада**: відновлюєш її — і вона стає серцем громади (стеля рівня всіх інших будівель).
3. Ти розширюєш село плитка-за-плиткою, ставиш **бензиновий Генератор** (живить електромережу й визначає, скільки будинків можуть бути "підключені"), призначаєш вцілілих на роботи, наймаєш героїв, відбиваєш нічні хвилі мутантів — а коли село стає фортецею, виходиш на карту Пустки (Phase 3+).

**MVP-ціль (Phase 1):** ~3 години однокористувацького геймплею — від "самотній сержант серед руїн" до "село з Сільрадою Lv2, запущеним Генератором, 4 героями, 6–8 будівлями і відбитими 3 хвилями мутантів".

## 1. Pillars (стовпи дизайну)

Усе, що ми робимо, має підсилювати один із цих стовпів. Якщо фіча не підсилює — ріжемо.

1. **"Кожен цвях — твій"** — відчуття, що гравець особисто підняв поселення з нуля. Кожна дія веде до видимої міні-перемоги/досягнення — прогрес відчутний на кожному кроці, без "порожніх" кліків.
2. **"Зона не прощає"** — ресурси скінченні, помилки коштують героїв, аномалії загрожують базі.
3. **"Люди важливіші за стіни"** — герої з іменами, біографіями, перками. Їх смерть болить.
4. **"Мапа розкривається поступово"** — як у *Tiles Survive!*, перші години ти бачиш тільки околицю табору. Великий світ — нагорода за прогрес.
5. **"Від табору до імперії"** — кожна фаза гри дає нову шкалу (одна база → регіон → Пустка). Механіки першої фази не викидаються, а масштабуються.

## 2. Фази розробки (ДУЖЕ ВАЖЛИВО)

Розробка поділена на **чотири фази**. Кожна фаза — **повноцінна гра сама по собі**. Якщо наступна не злетить — попередня все одно грабельна і продавана.

```
Phase 1 (MVP)       Phase 2            Phase 3             Phase 4 (опція)
[ Single-player ]  [ Кампанія,       ] [ Глобальна карта, ] [ Мультиплеєр,
[ одне поселення]  [ розширення      ] [ фракції,         ] [ альянси,
[ ~3 год        ]  [ героїв          ] [ 4X SP            ] [ PvP            ]
```

### Phase 1 — "Я, село, мутанти" (MVP) ← МИ ТУТ
- Одна локація: покинуте **село Глуш** у лісовому краях Пустки.
- Стартовий герой **Сержант Дзьоб** + 3–4 NPC, що приходять по сюжету.
- База на тайлах (~20×20): **Сільрада** (серце села), **Генератор** (енергія), Бар, Склад, Грядки, Будинок героя, Барак вцілілих, Майстерня, Лазарет, Стіна, Вежа — див. §4.4.
- Ресурси: 5–7 штук, **список на обговоренні** (§4.5).
- Призначення вцілілих на роботи (лісоруб / мисливець / огородник / кухар / інженер).
- Хвилі мутантів ~кожні 3–5 хв + текстові вилазки в околиці.
- **Автобій у реальному часі:** герої самі стріляють, гравець керує позиціями і здібностями.
- **Механіка відновлення** (§4.10): герої після довгих вилазок потребують `Горілку` або `Антирад` для повернення у боєздатний стан.
- Сюжет: 6–8 коротких діалогових сцен.

### Phase 2 — "Село росте"
- До 10 героїв, дерево перків кожному.
- **Рідкісні тири героїв (A / S / SS)** — аналог з *Tiles Survive!* (паркуємо, деталі уточнюємо на Phase 2):
  - **A** — звичайні (2 скіли: автоатака + 1 пасив)
  - **S** — рідкісні (4 скіли: автоатака + актив + 2 пасиви, останній розблоковується при макс-прокачці)
  - **SS** — легендарні, як S + покращені показники
- Apgrade будівель до Lv3, нові типи (радіовежа, лабораторія, гараж).
- Експедиції стають **окремими тактичними сценами** (не лише текст).
- 2 фракції з контрактами (**Порядок**, **Стерв'ятники**).
- Дослідження (технологічне дерево в лабораторії).

### Phase 3 — "Світова карта"
- Глобальна карта Пустки (гекси/тайли регіонів).
- 5 фракцій з AI-дипломатією (репутація, союзи, війни).
- Загони ходять по карті, беруть точки інтересу, б'ються з AI.
- Глобальні події ("Вибух", "Нашестя").
- **Все ще single-player**, опонент — AI.

### Phase 4 — "Альянси" (опціональна, ризикова)
- Онлайн-сервер (Godot multiplayer + бекенд, напр. Nakama).
- Альянси гравців, спільні війни, боси альянсу, арена 3×3 (як у *Tiles Survive!*).
- Сезони (~3 місяці), ресет, лідерборди.
- **Рішення про Phase 4 ухвалюємо не раніше, ніж Phase 3 буде в Steam** і покаже, що гра комусь потрібна.

⚠️ **Чому так.** Мультиплеєр з альянсами — це сервер, античіт, модерація, економіка, ~24/7 підтримка. Це окрема професія. Ми її робимо ОСТАННЬОЮ, а не ПЕРШОЮ, і тільки якщо Phase 1–3 показали життєздатність.

## 3. Core Gameplay Loop (Phase 1)

```
┌─ День (~5 хв) ─────────────────────┐
│ 1. Призначити героїв на роботи     │
│ 2. Розширити територію (відкрити   │
│    нову плитку = витрата ресурсу)  │
│ 3. Побудувати / апгрейднути        │
│ 4. Діалог / сюжетна сцена (іноді)  │
└─────────────┬──────────────────────┘
              │ настає ніч
              ▼
┌─ Ніч (~2 хв): хвиля мутантів ──────┐
│ герої на позиціях, автобій         │
│ гравець керує здібностями,         │
│ перекидає героїв між флангами      │
└─────────────┬──────────────────────┘
              │ хвиля відбита
              ▼
     нагороди, поранення, ремонт → День
```

**Ритм:** 70% "мирний час" (будуєш, призначаєш, читаєш), 30% "напруга" (бій). На відміну від survivor — там 95% бій.

## 4. Ключові механіки (дистилят *Tiles Survive!* + свої фішки)

### 4.1. Сільрада — серце села (+ Генератор як окрема вісь)

У нас **дві центральні будівлі** з різними ролями. Це свідомий відхід від *Tiles Survive!*, де Power Plant робить і те й інше.

**Сільрада** (адмінбудівля):
- Центральна клітинка мапи (2×2), відновлюється з руїн на початку гри.
- **Рівень Сільради = стеля рівня всіх інших будівель** (ключова вертикальна вісь прогресії).
- Вміщує сюжетні івенти, приймає контракти від фракцій (Phase 2+).
- Без палива. Апгрейд — будматеріалами, часом, **артефактами** (для старших рівнів).

**Генератор** (бензиновий, окрема будівля, 2×2):
- Виробляє **енергію** і роздає її "підключеним" будівлям (вежа, майстерня, лазарет, бар, лампи на стінах).
- **Рівень Генератора = скільки слотів електромережі доступно.** Lv1 = 3 слоти, Lv2 = 6, Lv3 = 10 (цифри чорнові).
- Потребує пальне: **Пальне** (бензин/солярка). Скінчилося → світло гасне, вежі мовчать, мораль падає, але *будівлі не ламаються*.
- Гравець сам вирішує, **які** будівлі "підключати" — це тактика (пріоритети між обороною і комфортом).

### 4.2. Плитки (tiles) і розширення території
- Карта — сітка ~20×20 плиток.
- На старті відкрито ~3×3 навколо Сільради.
- Нові плитки **відкриваються** ресурсом (деревина/метал) + часом "розчищення".
- Кожна плитка містить **одне з:** пусто / ресурс (ліс, металобрухт, ящик) / ворог (1–2 мутанти, треба зачистити) / аномалія (заблоковано) / сюжетна точка.
- Це **ключовий відрив** від survivor-дизайну: ти не бігаєш, ти **розкриваєш** карту.

### 4.3. Вцілілі та герої

**Два рівні юнітів:**

- **Вцілілі** (безіменні, до 20) — призначаються на роботи:
  - Лісоруб (генерує деревину)
  - Мисливець (провіант)
  - Збирач брухту (метал)
  - Кухар (конвертує провіант → готова їжа, мораль)
  - Інженер (прискорює будівництво)
- **Герої** (до 8, з іменами) — ідуть у бій та експедиції, мають перки.

Phase 1 ростер героїв:

| # | Кодове ім'я | Коли приєднується   | Роль       | Перк                                            |
|---|-------------|---------------------|------------|-------------------------------------------------|
| 1 | **Дзьоб**   | Пролог              | Лідер      | +10% ефективність усіх робіт у таборі           |
| 2 | **Кропива** | День 1, сюжет       | Розвідниця | Швидкість, бачить ворогів у тумані війни        |
| 3 | **Док**     | День 2, сюжет       | Медик      | Лікує в бою, відкриває лазарет                  |
| 4 | **Шип**     | День 3, сюжет       | Стрілець   | Бонусний урон по великих цілях                  |
| 5 | **Лом**     | День 4, опційно     | Інженер    | +20% до швидкості будівництва, ремонт у бою     |

### 4.4. Село (будівлі MVP)

> Розмір — у плитках. "Струм" = чи потребує слот електромережі від Генератора. "Lv стеля" = що обмежує апгрейд.

| Будівля                   | Розмір | Lv стеля    | Струм? | Функція                                                        |
|---------------------------|--------|-------------|--------|----------------------------------------------------------------|
| **Сільрада**              | 2×2    | —           | —      | Серце села. Стеля Lv усього. Сюжет, контракти фракцій.         |
| **Генератор**             | 2×2    | = Сільрада  | —      | Дає енергію. Lv = кількість слотів електромережі. Жере Пальне. |
| **Склад**                 | 2×2    | = Сільрада  | ні     | Ліміт на зберігання ресурсів. Lv = більший кап.                |
| **Барак вцілілих**        | 2×2    | = Сільрада  | ні     | Житло для безіменних вцілілих. +4 слотів за Lv.                |
| **Будинок героя**         | 1×1    | Lv3         | так    | Персональне житло іменного героя. +1 слот героя, маленький бонус до перка при "Lv2/3 будинку". |
| **Бар "Глуш"**            | 2×1    | = Сільрада  | так    | Конвертує `Провіант` → мораль; прискорює **відновлення** героїв (§4.10). |
| **Грядки**                | 2×2    | Lv3         | ні     | Пасивно генерують `Провіант` (повільніше за мисливця, але стабільно). |
| **Ліс навколо села**      | плитки | —           | —      | Плитки типу "ліс" поза забудовою. Лісоруб працює тут. Повільно **відновлюються** при невикористанні. |
| **Майстерня**             | 2×2    | = Сільрада  | так    | Крафт патронів, зброї, простих ліків.                          |
| **Самогонний апарат**     | 1×1    | Lv3         | так    | Виробляє `Горілку` з `Провіанту` (див. §4.10). Опційно — прибудова до Бару. |
| **Лазарет**               | 2×1    | = Сільрада  | так    | Лікує HP героїв після бою.                                     |
| **Стіна**                 | 1×1    | Lv3         | ні     | Блокує прохід ворогу, HP.                                      |
| **Ворота**                | 1×2    | Lv3         | ні     | Вхід, HP, можна замкнути.                                      |
| **Вежа**                  | 1×1    | = Сільрада  | так    | Автостріляє, потребує `Патрони`. Без струму → не стріляє.      |

**Логіка "струму":** вежа/бар/лазарет/майстерня мають бути вручну "підключені" до Генератора через панель. Якщо слотів Генератора менше ніж потрібних будівель → гравець сам пріоритезує.

### 4.5. Ресурси (ЧОРНОВА ТАБЛИЦЯ — НА ОБГОВОРЕННЯ)

> ⚠️ Це **не фінальний** список. Мета — не перевантажити гравця. Ціль: **5–7 ресурсів**, не більше.

| Ресурс       | Джерело                                    | На що витрачається                                | Статус          |
|--------------|--------------------------------------------|---------------------------------------------------|-----------------|
| `Деревина`   | Лісоруб на плитці лісу                     | Стіни, Lv1-будівлі, розчищення плиток             | ✅ beзумовно    |
| `Метал`      | Збирач брухту на плитці руїн               | Вежі, зброя, Lv2+                                 | ✅ безумовно    |
| `Провіант`   | Мисливець, грядки, консерви з руїн         | Утримання героїв і вцілілих, ввід у Бар           | ✅ безумовно    |
| `Патрони`    | Крафт у майстерні + знахідки               | Постріли в бою                                    | ✅ безумовно    |
| `Пальне`     | Експедиції, торговець                      | Годує Генератор                                   | ✅ безумовно    |
| `Горілка`    | Самогонний апарат (з Провіанту)            | Відновлення героїв після вилазок (§4.10)          | ✅ безумовно    |
| `Гроші/Хабар`| Продаж артефактів, контракти                | Найм, торгівля з NPC                              | 🟡 **під питанням** — чи треба в Phase 1, чи досить бартеру? |
| `Енергія`    | Генератор (pass через цикл)                | Живить підключені будівлі                         | ⚙️ derived — **не зберігаємо**, це потік |
| `Антирад`    | Експедиції, торговець, Phase 2 крафт       | Альтернатива Горілці — **швидше** відновлення     | 🟡 можливо об'єднати з Горілкою в один "recovery" ресурс |

**Питання на обговорення:**
1. Чи потрібен `Гроші/Хабар` окремо, чи краще в Phase 1 усе через бартер (Метал/Провіант як валюта), а `Гроші` вводимо лише в Phase 2 з фракціями?
2. Чи об'єднати `Горілка` + `Антирад` в один ресурс `Медикаменти` (простіше) — чи лишити два (Горілка = дешева/довга, Антирад = дорога/швидка)?
3. Чи виносити `Енергія` в UI взагалі, якщо це не stockpile, а тільки потік? (Мабуть, показувати лише "X/Y слотів Генератора зайнято".)

### 4.6. Автобій у реальному часі
- Хвилі мутантів атакують з 1–2 напрямків (позначено перед хвилею).
- Перед хвилею гравець **розставляє героїв на позиції** (пауза активна).
- Під час бою:
  - Герої **автостріляють** найближчу ціль.
  - Вежі стріляють, поки є енергія і патрони.
  - Гравець може перетягнути героя на іншу позицію, активувати **здібність** (кд), наказати відступ у лазарет.
- **Немає прямого контролю руху/стрільби** — це не шутер.

### 4.7. Експедиції (Phase 1 — текстові)
- Вибираєш 1–3 героїв, відправляєш на сусідній тайл "диких земель".
- Абстрактна подія: таймер → розіграш подій → повернення з нагородою/пораненням.
- Текстовий івент з 2–3 вибором ("Зустріли ходока. Напасти / Обмінятися / Обійти?").
- Шанси і нагороди залежать від перків героя.
- Після повернення з **довгих** вилазок герої отримують статус `Виснаження`/`Зараження` — див. §4.10.

### 4.8. Час
- Умовний день: **~7 хв** (5 хв день + 2 хв ніч).
- День: мирна фаза, роботи, будівництво.
- Ніч: хвиля.
- **Паузу** можна ставити в будь-який момент (SP-гра).

### 4.9. Сюжет і діалоги
- Твіновий стиль: портрет + текст + 2–4 варіанти відповіді.
- Ключові сцени: пролог, прибуття кожного героя, перший контакт з фракцією (День 3), телеграф Phase 2 (День 7).
- Туторіал **вшитий у сюжет**: нові механіки даються разом з репліками героїв.

### 4.10. Відновлення героїв після вилазок

**Проблема дизайну:** без "податку" на експедиції гравець буде ганяти героїв нон-стоп, не буде відчуття "Зона виснажує".

**Рішення — статуси `Виснаження` / `Зараження`:**

- Будь-яка експедиція **довша за N хвилин** ігрового часу (або в "гарячу" плитку з аномалією) → герой повертається з одним із статусів:
  - `Виснаження` — накопичувальний, знижує урон/швидкість; лікується **відпочинком** (час у Барі/Будинку героя) або `Горілкою` (швидше).
  - `Зараження` — аномальне/радіаційне; не лікується відпочинком, тільки `Антирадом` (або довгим курсом у Лазареті з `Медикаментами`).
- Герой зі статусом **може йти в бій**, але неефективно, і ризикує померти від "переповнення" (якщо 3+ рівні статусу).
- `Горілка` — дешева, повільна, з побічкою (-мораль на день). `Антирад` — дорогий, швидкий, без побічки.

**Phase 4 ремарка (монетизація, якщо колись буде):** `Антирад` — природний кандидат на "преміум-валюту" (прискорення). Але **в Phase 1–3 це чисто гральний ресурс**, знаходиться/крафтиться. Ніякого F2P-тиску.

## 5. Сеттінг і лор

### 5.1. Великий лор (загальні рамки)

- **Час дії:** близьке майбутнє, ~2050-ті. Конкретні країни/міста **не названі** — світ змінився до невпізнанності.
- **Як світ дійшов до цього:** серія локальних катастроф у 2030-х (ядерні інциденти + експерименти з аномальною фізикою) + коротка війна → **"Розлом"** — каскадна подія, після якої аномальні Зони почали **розростатися самі** й перекривати цілі континенти.
- **Сьогодні:** поверхня землі — це **Пустка**: плями аномалій, руїни, мутовані екосистеми. Людство живе в острівцях "чистих" земель — селах, бункерах, торгових містах. Централізованих держав немає. Є фракції.
- **Де відбувається Phase 1:** лісовий край Пустки, село **Глуш** (колишнє передмістя якоїсь колишньої агломерації). Назва умовна, рідна мова гри — **універсально-слов'янська**, без географічних прив'язок.
- **Тон:** похмурий, людяний. Не чорнушний. Чорний гумор у логах.
- **Мова:** інтерфейс і тексти — українська (основна), англійська (локалізація пізніше).
